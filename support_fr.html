Conseils pour l'analyse IA

  • Éviter les chevauchements: Veuillez filmer dans un endroit où vous ne chevauchez pas d'autres personnes.
  • Rester dans le cadre: Assurez-vous que tout votre corps reste dans l'angle de la caméra.
  • Miroirs: Minimisez les reflets dans les miroirs pour éviter les erreurs de détection.
